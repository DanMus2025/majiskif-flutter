# Scenario demo KESE

## Objectif

Produire deux videos de demonstration reelle:

- `KESE-demo-desktop.mp4`
- `KESE-demo-mobile.mp4`

Les deux videos doivent montrer le fonctionnement reel de KESE avec le backend cloud:

```text
https://majiskif-kese-cloud-api.onrender.com/api/v1/cloud
```

## Parcours recommande

1. Ouvrir KESE.
2. Afficher la page d'aide et les contacts.
3. Activer une licence ou ouvrir une session existante.
4. Verifier que l'adresse cloud utilisee est:
   `https://majiskif-kese-cloud-api.onrender.com/api/v1/cloud`
5. Entrer dans l'application.
6. Ouvrir les parametres de l'entreprise.
7. Ajouter ou verifier le logo de l'entreprise.
8. Creer une categorie de produits.
9. Ajouter deux produits:
   - Riz local 25 kg
   - Huile vegetale 5 L
10. Ajuster le stock initial.
11. Ouvrir la page Vendre.
12. Ajouter les produits au panier.
13. Choisir ou creer un client.
14. Valider une vente payee.
15. Ouvrir le ticket ou la facture.
16. Montrer la previsualisation.
17. Montrer les boutons PDF / impression si disponibles.
18. Ouvrir Caisse.
19. Montrer le resume des ventes, benefices et paiements.
20. Ouvrir Produits / Stock.
21. Verifier la diminution du stock.
22. Ouvrir Messages.
23. Envoyer un court message interne.
24. Ouvrir Plus.
25. Lancer Synchroniser.
26. Montrer le message de confirmation.
27. Fermer l'application proprement.
